Elephant Listening Project

This project aims to protect the the forest elephants from poaching in the African forests, as they may go extinct in 10 years The goal was to develop an algorithm that can detect gunshot sounds by using machine learning, software engineering, computer vision and cloud computing techniques. The model is going to be used in on detectors that will be parachuted in the forests so that ecoguards can see when a gunshot was detected and go catch the poachers.

Mobility-impaired people

The initial brief for this project was finding a socioeconomic or environmental problem and use science and technology to provide a solution. The problem our group developed was accessibility of technology for the mobility impaired. The particular problem chosen was to allow people with severe mobility impairments easy and effcient access all functionalities to the PC. The solution our group developed used the established technologies of speech-to-text and sip-and-puff to effectively implement alternative interfaces for typing and mouse cursors and clicks. The aim was to create a complete solution integrating both cursors and text input in one solution. Furthermore, the goal was to make a solution which was cheaper as a lot of the existing solutions are expensive. The speech to text was used to input text but also access shortcuts usually accessed from a keyboard. The sip-and-puff together with joystick were used to implement cursor movement and different types of `clicks'. The product produced ended up being a signifficantly cheaper way to interface with a computer for those with signifficant mobility impairments compared to competitors on the market. It also provided a complete solution to access all functionalities on the computer. Given the constraints of the project, a prototype was completed which illustrates all the functionalities of desired product.
